# 📱 Persistência de Dados com React Native

## 🎯 Sobre o Projeto

Projeto desenvolvido para estudos de persistência de dados em aplicações React Native utilizando TypeScript.

O objetivo é explorar conceitos como armazenamento local, AsyncStorage, manipulação de dados, operações assíncronas e boas práticas de desenvolvimento mobile.

🚧 Projeto em desenvolvimento e evolução contínua.

## 🛠️ Tecnologias Utilizadas

* React Native
* Expo
* TypeScript
* AsyncStorage
* Node.js

## 📚 Tópicos Estudados

* Persistência de dados
* AsyncStorage
* Async/Await
* Promises
* Serialização com JSON
* Gerenciamento de estado
* Boas práticas de desenvolvimento

## 🚀 Executando o Projeto

```bash
git clone https://github.com/PamelaMBadermann/data-persistence-with-react-native.git

cd data-persistence-with-react-native

npm install

npx expo start
```

Este projeto faz parte dos estudos desenvolvidos durante a graduação em Engenharia de Software e tem como propósito consolidar conhecimentos em persistência de dados no desenvolvimento mobile.
